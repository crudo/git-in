# git-in

[![Greenkeeper badge](https://badges.greenkeeper.io/crudo/git-in.svg)](https://greenkeeper.io/)
Interactive Git File Restore

![demo](demo/git-in.gif)

### Requirements
* Node >= 4

### Install
```bash
$ npm install -g git-in
```

### Use
```bash
$ git-in
```

### Help
```bash
$ git-in --help
```
